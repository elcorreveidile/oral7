# Ejemplos de Registros Contrastados

## Diferencias entre lenguaje formal e informal en español C1

---

## 1. Introducción a los registros

### ¿Qué es el registro lingüístico?

El **registro** es el nivel de formalidad que usamos según el contexto, la relación con el interlocutor y la situación comunicativa.

### Tres niveles principales

| Nivel | Uso típico | Características |
|-------|------------|-----------------|
| **Formal** | Académico, profesional, presentaciones | Vocabulario preciso, estructuras complejas, "usted" |
| **Neutro** | Conversación educada, semi-formal | Mezcla equilibrada, register moderado, "tú/usted" según contexto |
| **Informal** | Amigos, familia, situaciones casuales | Muletillas, contracciones, "tú", expresiones coloquiales |

---

## 2. Contrastes por categoría gramatical

### 2.1 Pronombres de tratamiento

| Formal | Informal |
|--------|----------|
| **Usted** / **Ustedes** | **Tú** / **Vosotros** (España) / **Ustedes** (Latinoamérica) |
| "¿**Usted** podría...?" | "¿**Tú** podrías...?" |
| "A **usted** le interesa" | "A **ti** te interesa" |

**En contextos académicos/profesionales:**
- Use **usted** con:
  - Profesores
  - Superiores jerárquicos
  - Personas mayores
  - Desconocidos en contexto formal

- Puede usar **tú** con:
  - Compañeros de igual nivel
  - Situaciones menos formales
  - Cuando la otra persona inicia el tuteo

---

### 2.2 Formas verbales de tratamiento

| Formal (Usted) | Informal (Tú) |
|----------------|---------------|
| **Usted es** | **Tú eres** |
| **Usted tiene** | **Tú tienes** |
| **Usted va** | **Tú vas** |
| **Usted esté** (subjuntivo) | **Tú estés** (subjuntivo) |
| **Para usted** | **Para ti** |
| **Con usted** | **Contigo** |
| **De usted** | **De ti** |

### 2.4 Conectores por registro

| Formal | Informal |
|--------|----------|
| **Sin embargo** | **Pero** |
| **No obstante** | **Pero** / **Aunque** |
| **Por lo tanto** | **Así que** / **Por eso** |
| **En consecuencia** | **Entonces** |
| **Asimismo** | **También** |
| **Por añadidura** | **Además** |
| **En primer lugar** | **Primero** |
| **En última instancia** | **Al final** |
| **En definitiva** | **En resumen** / **O sea** |
| **A pesar de ello** | **Aunque** |

---

### 2.5 Expresiones de opinión

| Formal | Informal |
|--------|----------|
| **Considero que**... | **Creo que**... / **Me parece que**... |
| **Estimo que**... | **Pienso que**... |
| **Opino que**... | **Para mí**... |
| **Desde mi punto de vista**... | **Para mí**... / **Yo digo que**... |
| **En mi opinión**... | **Yo creo que**... |
| **A mi juicio**... | **Yo diría que**... |
| **Me inclino a pensar que**... | **Yo diría que**... |
| **Sostengo que**... | **Digo que**... |

---

### 2.6 Muletillas vs. conectores

| Muletilla informal (Evitar) | Conector formal (Usar) |
|------------------------------|----------------------|
| **O sea**... | **Es decir**... / **En otras palabras**... |
| **Entonces**... (repetitivo) | **Por lo tanto**... / **En consecuencia**... |
| **O sea, entonces**... | **Es decir, por lo tanto**... |
| **¿Sabes?** / **¿Sabes?** | *(pausa)* o **¿Comprende?** (con moderación) |
| **Es que**... | **La razón es que**... / **Dado que**... |
| **Como que**... | **En cierto modo**... |
| **Resulta que**... | **El caso es que**... / **Sucede que**... |

## 3. Ejemplos de contrastes por contexto

### 3.1 Contexto académico

**Formal (Presentación de conferencia):**
> "En primer lugar, me gustaría **exponer** los principales hallazgos de mi investigación. **Posteriormente**, **analizaremos** las implicaciones teóricas. **Por consiguiente**, **concluiremos** con una discusión sobre las aplicaciones prácticas."

**Informal (Charla con compañeros después de clase):**
> "Bueno, primero les **cuento** lo que **encontré**. Después, lo **vemos** y **hablamos** de qué significa. Y al final, **platicamos** de cómo se puede usar."

---

### 3.2 Contexto profesional

**Formal (Entrevista de trabajo):**
> "**Agradezco** la oportunidad de **presentar** mi candidatura. **Considero** que mi **experiencia profesional** en el área me **permite aportar valor** a su organización. **Estoy convencido de** que mi **perfil se ajusta** a las **necesidades del puesto**."

**Informal (Hablando con amigos sobre el trabajo):**
> "**Oye**, me dieron la oportunidad. **Creo** que con lo que **sé** les puedo **ayudar**. **Estoy seguro** de que **voy bien** para el puesto."

---

### 3.3 Contexto de negociación

**Formal:**
> "**Desearía manifestar** que **nuestra posición** es **sustancialmente diferente**. **No obstante**, **estamos dispuestos a considerar** una **solución intermedia** que **satisfaga los intereses de ambas partes**."

**Informal:**
> "**Quiero decir** que **lo que yo pienso** es **diferente**. **Pero**, puedo **aceptar algo en medio** que **nos sirva a los dos**."

---

### 3.4 Contexto de Complaint (reclamación)

**Formal (Carta de reclamación):**
> "**Deseo expresar mi disconformidad** con el **servicio recibido**. **Solicito** que se **subsanen** las **irregularidades detectadas** en la **mayor brevedad posible**."

**Informal (Queja en persona):**
> "**Oye**, estoy **molesto** con lo que **pasó**. **Quiero que lo arreglen** **ya**."

| España | Latinoamérica |

## 5. Estrategias para cambiar de registro

### De informal a formal (elevando el registro)

1. **Reemplazar verbos básicos**
   - Hacer → Realizar / Efectuar
   - Dar → Proporcionar / Otorgar
   - Decir → Manifestar / Expresar

2. **Usar conectores formales**
   - Pero → Sin embargo / No obstante
   - Y → Asimismo / Además
   - Entonces → Por lo tanto / En consecuencia

3. **Añadir estructuras gramaticales complejas**
   - "Es posible que..." + subjuntivo
   - "Cabe destacar que..."
   - "Es importante considerar..."

4. **Eliminar muletillas**
   - Quitar: "o sea", "es que", "¿sabes?"
   - Reemplazar por: pausas, "es decir", "en otras palabras"

---

### De formal a informal (bajando el registro)

1. **Simplificar verbos**
   - Realizar → Hacer
   - Solicitar → Pedir
   - Manifestar → Decir

2. **Usar conectores simples**
   - Sin embargo → Pero
   - Por consiguiente → Así que / Entonces
   - Asimismo → También

3. **Añadir expresiones coloquiales**
   - "Oye..." / "Mira..."
   - "¿Sabes?"
   - "Claro", "Obvio"

4. **Contracciones y elisiones**
   - Para + tú → Pa' ti (muy informal)
   - Voy a me voy → Me voy

## 7. Adapting register to audience

### Checklist antes de hablar:
- [ ] ¿Quién es mi audiencia? (profesores, amigos, colegas)
- [ ] ¿Cuál es el contexto? (presentación, café, entrevista)
- [ ] ¿Qué nivel de formalidad es apropiado?
- [ ] ¿Debo usar "tú" o "usted"?
- [ ] ¿Qué conectores son más adecuados?

### Errores comunes:
- ❌ **Tuteo excesivo** en contexto formal
- ❌ **Usted** con amigos cercanos (puede crear distancia)
- ❌ **Muletillas** ("o sea", "es que") en discurso académico
- ❌ **Vocabulario overly formal** en contextos casuales (suena pretencioso)
