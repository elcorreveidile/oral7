# PRUEBA DE ACCESO A RESULTADOS

## Enlaces correctos:

### Para estudiantes (votar):
http://localhost:3000/votacion-debates

### Para profesor (ver resultados):
http://localhost:3000/admin/debates/resultados

## Diferencia importante:

- ✅ `/admin/debates/resultados` = Página web visual
- ❌ `/api/debates/results` = Solo JSON (no es una página)

## Si sigues viendo JSON:

Es posible que estés accediendo a la API directamente. Asegúrate de usar:
http://localhost:3000/admin/debates/resultados

Con slashes "/", no con "/api/"
