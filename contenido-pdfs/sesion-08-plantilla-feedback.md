## FRASES ÚTILES PARA DAR FEEDBACK

### Para destacar lo positivo:
- "Destacaste especialmente en..."
- "Me gustó cómo usaste..."
- "Tu argumento sobre ___ fue muy convincente"
- "El conector que usaste (___) fue perfecto"

### Para sugerir mejoras:
- "Podrías mejorar..."
- "Sería útil practicar más..."
- "Te sugiero que..."
- "Para la próxima, intenta..."

### Para objetivos específicos:
- **Muletillas**: "Intenta reemplazar 'o sea' por 'es decir' o 'en otras palabras'"
- **Conectores**: "Podrías usar más conectores de contraste como 'sin embargo' o 'no obstante'"
- **Registro**: "En contextos formales, evita el tuteo"
