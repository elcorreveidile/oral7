## 🎯 Niveles MCER

| Nivel | MCER | Objetivo | Situaciones | Características |
|-------|------|----------|-------------|-----------------|
| **A1** | Acceso | Supervivencia básica | 6 | Frases 2-4 palabras, vocabulario visual |
| **A2** | Plataforma | Comunicación simple | 6 | Tareas cotidianas, 10-12 líneas |
| **B1** | Intermedio | Independencia cotidiana | 3 | Situaciones familiares, 10 líneas |
| **B2** | Intermedio Alto | Fluidez profesional | 3 | Vocabulario técnico, formal |

## 🚀 Uso Rápido

### Para profesores

1. Leer `GUIA_PROFESORES.md`
2. Seleccionar nivel según grupo
3. Usar PDFs en clase (vocabulario + ejercicios)
4. Reproducir MP3s (línea por línea o completo)
5. Practicar role-play

### Para estudiantes autodidactas

1. Hacer test de nivelación correspondiente
2. Empezar por nivel determinado
3. Escuchar diálogo sin texto
4. Seguir con transcripción
5. Resolver ejercicios
6. Practicar repetición en voz alta

## 📈 Progresión Recomendada

### Curso completo (20 semanas)

```
Semanas 1-4:   Nivel A1 (6 situaciones) → Fundamentos
Semanas 5-8:   Nivel A2 (6 situaciones) → Comunicación básica
Semanas 9-11:  Nivel B1 (3 situaciones) → Fluidez cotidiana
Semanas 12-14: Nivel B2 (3 situaciones) → Contextos profesionales
Semanas 15-17: Nivel C1 (4 situaciones) → Competencia académica
Semanas 18-20: Nivel C2 (4 situaciones) → Maestría lingüística
```

### Por sesión

**Primera sesión (50-60 min):**
- Presentación de vocabulario
- Primera escucha
- Comprensión global

**Segunda sesión (50-60 min):**
- Segunda escucha con texto
- Ejercicios de comprensión
- Role-play inicial

**Tercera sesión (30-45 min):**
- Role-play completo
- Producción escrita
- Evaluación

## 🔧 Instalación y Uso

### Requisitos
- macOS (para usar `say`)
- o Reproductor de MP3/PDF (cualquier dispositivo)

### Uso local (macOS)
```bash
# Generar nuevo diálogo
./scripts/generar-dialogo.sh "restaurante"

# Reproducir línea específica
say -v "Mónica" "¡Buenos días!"

# Convertir a MP3
ffmpeg -i archivo.aiff archivo.mp3
```

### Uso web (Next.js)
```bash
# API endpoint disponible
POST /api/dialogos/generar
{
  "situacion": "restaurante",
  "usarIA": false
}
```
